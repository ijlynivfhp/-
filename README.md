# 简易框架
简易开发框架，完全可以丢弃动软代码生成器的开发模式， 曾经陪伴多年的三层架构或者说工厂模式什么的。
亮点：
1. asp.net core；
2. dapper;
3. Repositories | CRUD for Dapper
4. 轻量级orm POCO
5. 关注业务逻辑代码

使用：
1 建立数据库
2 修改连接字符串
